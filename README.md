# Zombera

Unity game architecture scaffold for a zombie survival squad game.

## Current State

- Universal unit architecture (player, squad, survivor, zombie)
- Event-driven manager layer
- World/chunk scaffolding
- Inventory, loot, combat, base building, recruitment skeletons

## Repository Notes

- This repo tracks source architecture and design docs.
- Unity-generated folders are ignored via `.gitignore`.

## Next Step

Continue implementing systems incrementally (starting from the existing character and gameplay flow work).
